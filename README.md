# GeneiRyodan_SDG_Project

Project Title: AgriBridge: A Standalone Decision System for Precision Rice Management in Connectivity-Independent Zones

Description: Rice farmers in Nueva Ecija, Philippines, currently face a 35–42% yield gap caused by timing-based inefficiencies in irrigation and fertilization that traditional farming calendars can no longer address. AgriBridge is an Offline-First Progressive Web App (PWA) designed to bridge this gap by providing precise, data-driven prescriptions for crop management without requiring a stable internet connection. By processing local environmental data on entry-level hardware, the system helps smallholders reduce fertilizer waste by 20% and identify ideal harvest windows with 90% accuracy. This project directly advances UN Sustainable Development Goals 2 (Zero Hunger), 8 (Decent Work and Economic Growth), and 12 (Responsible Consumption and Production) by democratizing precision agriculture for rural communities.
